   |   |                  Início da Partida:                      |   |   
   |   |                                                           |   |   
   |   | A partida começa com todos os jogadores lançando o dado. |   |   
   |   | O jogador com o maior número inicia o jogo,              |   |   
   |   | seguindo-se o sentido horário.                           |   |   
   |   |                                                           |   |   
   |   |                   Jogabilidade:                           |   |   
   |   |                                                           |   |   
   |   | Utiliza-se um dado para determinar os movimentos.         |   |   
   |   | Avanços conforme os pontos do dado.                      |   |   
   |   | '1' ou '6' permitem mover um peão da casa de partida.    |   |   
   |   | Ao tirar '6', o jogador pode lançar novamente.           |   |   
   |   |                                                           |   |   
   |   |               Número de Jogadores:                        |   |   
   |   |                                                           |   |   
   |   | 2, 3 ou 4 jogadores podem competir.                     |   |   
   |   |                                                           |   |   
   |   |                 Tabuleiro e Peças:                        |   |   
   |   |                                                           |   |   
   |   | Tabuleiro quadrado, percurso em cruz.                    |   |   
   |   | Cada jogador possui quatro peões iniciando na base da    |   |   
   |   | mesma cor.                                                |   |   
   |   |                                                           |   |   
   |   |                Objetivo do Jogo:                          |   |   
   |   |                                                           |   |   
   |   | Levar os quatro peões ao ponto final de sua cor primeiro.|   |   
   |   |                                                           |   |   
   |   |              Movimentação dos Peões:                      |   |   
   |   |                                                           |   |   
   |   | Movimento no sentido horário.                             |   |   
   |   | '6' necessário para iniciar ou mover da base.            |   |   
   |   | Movimento determinado pelo dado.                          |   |   
   |   | '6' permite lançar novamente.                            |   |   
   |   |                                                           |   |   
   |   |                 Captura de Peões:                        |   |   
   |   |                                                           |   |   
   |   | Peões adversários retornam à base se capturados.         |   |   
   |   | Dois peões da mesma cor na mesma casa são seguros.       |   |   
   |   |                                                           |   |   
   |   |               Chegada ao Ponto Final:                    |   |   
   |   |                                                           |   |   
   |   | Após uma volta completa, avance pela reta final.         |   |   
   |   | Chegada exige número exato no dado.                      |   |   
   |   | Excedendo, avance e retorne, aguardando a próxima        |   |   
   |   | jogada.                                                   |   |   
   |   |                                                           |   |   
   |   |                         Vitória:                         |   |   
   |   |                                                           |   |   
   |   | Primeiro jogador a levar seus quatro peões ao ponto      |   |   
   |   | final de sua cor vence.                                  |   |
